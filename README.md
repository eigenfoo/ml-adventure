# ML Adventure

## Requirements

- Python 3
- [`sl`](https://github.com/mtoyoda/sl) package
- `pgrep` UNIX utility

## Setup

```
git clone https://github.com/eigenfoo/ml-adventure.git
cd ml-adventure/
pip install -r requirements.txt
cd mini_canne/corpora/
./download_frames.sh
cd ../../
```

## Usage

```
python game.py
```
