# ML Adventure

A text-based adventure game that uses a neural network to generate audio!

## Requirements

- Python 3
- `pgrep` UNIX utility

## Setup

```
git clone https://github.com/eigenfoo/ml-adventure.git
cd ml-adventure/
pip install -r requirements.txt
cd mini_canne/corpora/
./download_frames.sh
cd ../../
```

## Usage

```
python game.py
```
